Error IY4002: Missing Required Tag Field

This error occurs when a preprocessing tag is missing a required field.

COMMON CAUSES:
• Forgot to include mandatory field like 'template' in !$map
• Misspelled field name (e.g., 'tempate' instead of 'template')
• Wrong tag structure or syntax

EXAMPLES:

❌ Incorrect - missing template:
  result: !$map
    items: [1, 2, 3]
    # missing template field

❌ Incorrect - misspelled field:
  result: !$map
    items: [1, 2, 3]
    tempate: "{{item}}"

✅ Correct:
  result: !$map
    items: [1, 2, 3]
    template: "{{item}}"

REQUIRED FIELDS BY TAG:
• !$map: items, template
• !$if: condition, then
• !$let: bindings, template
• !$join: array, delimiter

• IY5001: Type mismatch in operation
