Error IY2001: Variable Not Found

This error occurs when you reference a variable that hasn't been defined in the current scope.

COMMON CAUSES:
• Typo in variable name
• Variable defined in different scope
• Missing $defs or $imports section
• Attempting to use CloudFormation variables in preprocessing context

EXAMPLES:

❌ Incorrect:
  result: !$ undefined_var

✅ Correct:
  $defs:
    my_var: "hello"
  result: !$ my_var

✅ Also correct:
  $imports:
    config: file:config.yaml
  result: !$ config.database_url

AVAILABLE SCOPES:
• $defs: Document-local definitions
• $imports: External data imports  
• Local variables: Loop variables in !$map, !$each
